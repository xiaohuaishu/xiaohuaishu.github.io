关于我

个人简介

从搜索到对话，从大模型到智能体——12年 AI 技术演进见证者与实践者。作为微软小冰初创成员，我见证并参与了中国对话式 AI 从萌芽到成熟的完整历程。在小米，我带领团队深耕大语言模型预训练与智能体系统研发，推动 AI 技术在真实场景中的落地应用。

我的研究方向聚焦于人机对话与大语言模型，致力于探索 AI Agent 在智能终端、游戏、巡检机器人等领域的应用边界与落地挑战。同时，我积极参与学术社区建设，担任 CCF YOCSEF 总部学术委员、CCF 大模型论坛执委等职务，促进学术界与产业界的深度交流。

研究方向

智能体系统

研究大模型驱动的 AI Agent 架构，包括规划、推理、工具调用、多模态交互等核心技术。探索智能手机 + AI Agent 的未来形态与技术挑战。

大语言模型

专注于大语言模型的预训练、微调、推理优化。在位置编码、KV Cache 压缩、长上下文处理等方面开展前沿研究。

人机对话

超过10年人机对话研究经验，从闲聊对话到任务型对话，探索对话系统的理解、生成、评估等关键问题。

GUI Agent

研究移动端 GUI Agent，构建页面到达与操作两阶段框架，提升 Agent 在复杂界面中的任务完成能力。

多模态智能

探索视觉-语言模型在多图像场景中的感知、理解和推理能力，研究焦点为中心的视觉链等新范式。

能力评估

研究大模型能力评价体系，探索能力边界探测框架，关注模型的泛化性、鲁棒性、可解释性等问题。

代表性论文

ReachAgent: Enhancing Mobile Agent via Page Reaching and Page Operation

NAACL 2025 主会长文

提出两阶段框架，将手机控制任务拆分为页面到达和操作子任务，使 Agent 更关注整体任务完成能力。任务级别准确率提升 4.72%。

Multilingual Machine Translation with Open Large Language Models at Practical Scale: An Empirical Study

NAACL 2025 主会长文

提出平行优先、单语次之（PFMS）训练策略，基于 Gemma2 训练的 GemmaX2-28 模型翻译性能达到 GPT-4 Turbo 水平。

HoPE: A Novel Positional Encoding Without Long-Term Decay for Enhanced Context Awareness and Extrapolation

ACL 2025 主会长文

提出高频旋转位置编码（HoPE），打破长期衰减限制，提升模型上下文感知能力与外推任务中的注意力鲁棒性。

TailorKV: A Hybrid Framework for Long-Context Inference via Tailored KV Cache Optimization

ACL 2025 Findings 长文

提出混合压缩方法，无缝集成量化和卸载。128k 上下文的 Llama-3.1-8B 可在单个 RTX 3090 GPU 上部署，解码速度 82ms/token。

ToolPlanner: A Tool Augmented LLM for Multi Granularity Instructions with Path Planning and Feedback

EMNLP 2024 主会长文

提出两阶段强化学习框架，利用路径规划和反馈机制增强 LLM 的任务完成和指令遵循能力，匹配率提升 26.8%。

MobileVLM: A Vision-Language Model for Better Intra- and Inter-UI Understanding

EMNLP 2024 Findings 长文

提出增强 UI 内和 UI 间理解的 VLM，构建 300 万 UI 页面的 Mobile3M 数据集，提升模型在移动端环境中的表现。
